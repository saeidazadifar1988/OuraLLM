# OuraLLM
Prompt-based framework for depression and anxiety prediction using Oura Ring data and pretrained LLMs -IEEE-EMBS BSN 2026


# Study description 
The proposed approach transforms physiological, behavioral, demographic, and anthropometric measurements into structured natural-language prompts and evaluates the effectiveness of in-context learning using open-weight large language models (LLMs).

The study uses wearable data from 1,285 participants in the Northern Finland Birth Cohort 1986 (NFBC 1986), including activity, sleep, heart rate (HR), and heart rate variability (HRV) metrics collected over 4–8 days in free-living conditions.

Features
Prompt-based wearable mental health prediction
Zero-shot, rule-based, 2-shot, and 4-shot prompting strategies
Evaluation of multiple open-weight LLMs:
Llama 3.1-8B
BioMistral-7B
Qwen2.5-7B
Baseline comparisons with:
Logistic Regression
SVM
Random Forest
XGBoost
LSTM
GRU
Structured wearable-to-text transformation pipeline
Experiments on short-horizon wearable sensing data
